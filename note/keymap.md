### 命令行
``` txt
Control + A 光标移动到行首
Control + E 光标移动到行尾
Control + U 清除当前行

Option + 左(自带终端) | Alt + B(iTerm2) 向前移动一个单词
Option + 右 | Alt + F 向后移动一个单词
移动单词的快捷键不起作用：
iTerm2: Profiles -> Defalut -> keys -> 选择Esc+
自带终端： 秒速文件 -> 默认 -> 键盘 -> 将Option键用作Meta键
```
