### 使用库

python-docxx

### 安装解析库

``` shell
$ pip3 install python-docx
```





### 关键含义

Document对象：将整个文章看作是一个Document对象

Paragraph对象：段落，存放在`document.paragraphs`中

Run对象：行内元素，存放在`paragraph.runs`中。
